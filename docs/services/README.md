# Backend Services Documentation

This directory contains documentation for all backend services in the dock108 monorepo.

- **[`theory-bets-scraper.md`](theory-bets-scraper.md)** - Sports data ingestion service (boxscores, odds)
- **[`data-workers.md`](data-workers.md)** - Celery workers for background tasks (YouTube cache, odds snapshots, prices)
